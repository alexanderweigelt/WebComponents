# HTML 5 Boilerplate

Create a static site or simple app.

## Building the application

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

To lint your code

```sh
npm run lint
```

## Running

```sh
node dist/assets/bundle.js
```
